# golang-cli-mysql-simple

## Description
A POC for connecting a database to golang.

## Tech stack
- bash
- golang 1.13

## Docker stack
- ubuntu:latest
- mariadb:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
